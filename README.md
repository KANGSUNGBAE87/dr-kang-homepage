# Dr. Kang Studio Homepage

Public static homepage for `닥터강 / Dr. Kang Studio`.

## Purpose

This site is a simple public company homepage that can be used for:

- Google Play developer organization profile review
- Google Search Console ownership verification
- Public introduction of Dr. Kang Studio projects

## Local Preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8788
```

## Search Console Verification

When Google Search Console gives an HTML verification file, place it in this
repository root and commit it.

Example:

```text
google0000000000000000.html
```

## Company Info Placeholders

Replace the placeholders in `index.html` after the legal/company profile is ready:

- Representative
- Business registration number
- Business address
